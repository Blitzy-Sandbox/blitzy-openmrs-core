# OpenMRS Core

OpenMRS is a patient-based medical record system focusing on giving providers a
free, customizable electronic medical record system (EMR).

The mission of OpenMRS is to improve health care delivery in resource-constrained
environments by coordinating a global community that creates a robust, scalable,
user-driven, open source medical record system platform.

This site documents the `openmrs-core` repository, which contains the Java
platform that powers OpenMRS. The platform is built with Maven and is deployed
as a WAR file (`openmrs.war`) into a servlet container such as Tomcat or Jetty.

## What You'll Find Here

- A repository tour describing the top-level Maven modules.
- Build, deploy and Docker workflows distilled from the project README.
- Pointers to upstream wiki and developer guides for deeper topics
  (modules, SDK, configuration).
